# Listados Web

Aplicacion web 100% cliente para cargar archivos CSV o XLSX y generar listados descargables en PDF o XLSX desde el navegador.

## Contenido del repositorio

- `web_listados.html`: aplicacion principal.
- `entrada/Datos_completos.csv`: archivo de ejemplo / base de carga.

## Uso

1. Abrir `web_listados.html` en un navegador moderno.
2. Cargar un archivo `.csv`, `.xlsx` o `.xls`.
3. Elegir el formato de salida `PDF` o `XLSX`.
4. Descargar la planilla correspondiente.

## Requisitos

- Navegador moderno con soporte para JavaScript, descargas locales y lectura de archivos.
- Conexion a internet para cargar las librerias externas desde CDN.

## Notas

- No requiere backend.
- El procesamiento ocurre en el navegador.
- Los archivos de salida generados localmente no forman parte del repositorio.
